# 小红书排版工具 Markdown 规范 - LLM Prompt

请按照以下规范将文案重新排版，确保可以直接复制到小红书排版工具使用：

## 格式规范：
- 主标题：`# 标题 😊` （一个#，可加emoji）
- 副标题：`## 副标题` （两个#，可选）
- 署名：`> 作者名` （>开头，可选）
- 正文：直接书写，段落间空行分隔
- 强制换行：`<br>` （段落内换行）
- 重点高亮：`**重点内容**` （会显示绿色高亮）
- 列表：`- 要点` （支持子列表缩进）
- 卡片分割：`---` （单独一行，分割不同主题卡片）

## 排版要求：
1. 主标题简洁有力（12字内）
2. 每个卡片主题独立，内容200字内
3. 合理使用**高亮**突出关键信息
4. 适度添加emoji增强表现力
5. 逻辑清晰，层次分明
6. 保持原文核心内容不变
7. 支持使用<br>标签进行段落内换行

## 示例格式：
```
# 主标题 🌟
## 副标题说明
> 作者名

正文第一段内容，**重点信息**用加粗突出。

正文第二段内容，<br>可以使用br标签换行。

---

## 卡片2标题 💡

卡片2的内容：
- 要点1
- 要点2
- **重点要点**

记住这个重要原则，<br>换行让内容更清晰。

---

## 卡片3标题 ✨

卡片3内容...
```
